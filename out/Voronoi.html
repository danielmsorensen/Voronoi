Voronoi(id, w, h, v)

Creates and draws a Voronoi diagram.
Parameters:
Name Type Description
id string The id of the element to display the chart inside.
w number The width of the chart.
h number The height of the chart.
v Array.<Object> The array of vertices to be drawn.
Source:
  • voronoi.js, line 23
Example
let id = "chart"; // <div id="chart"></div>
let w = 960;
let h = 500;

let vertices = d3.range(100).map(function(d) {
    return [Math.random() * w, Math.random() * h];
}

let v = new Voronoi(id, w, h, vertices);

(static) getVertices(text, columnSepopt, lineSepopt, xColumnopt, yColumnopt, overlapopt, addDataopt) → {Array.<Object>}

Select an array of vertices from a string in DSV format
Parameters:
Name Type Attributes Default Description
text string The data in DSV format.
columnSep string <optional>
 "," The delimiter that separates columns in text.
lineSep string <optional>
 "\n" The newline character that separates the rows in text.
xColumn number <optional>
 0 The column position for the x-axis of the vertices.
yColumn number <optional>
 1 The column position for the y-axis of the vertices.
overlap number <optional>
 0 The minimum distance squared between 2 points before they are merged.
addData boolean <optional>
 false Whether or not to include the other data along with the vertex coordinates.
Source:
  • voronoi.js, line 171
Returns:
An array of vertices selected from text.
Type
Array.<Object>
Examples
let text = "0,0;1,1;2,4;3,9;4,16;5,25";
// returns [[0, 0], [1, 1], [2, 4], [3, 9], [4, 16], [5, 15]]
getVertices(text, ",", ";", 0, 1);
let text = "point0,0,0;point1,1,1;point2,2,4;point3,3,9;point4,4,16;point5,5,25";
// returns [[0, 0], [1, 1], [2, 4], [3, 9], [4, 16], [5, 15]]
getVertices(text, ",", ";", 1, 2);
let text = "point0,0,0;point1,1,1;point2,2,4;point3,3,9;point4,4,16;point5,5,25";
// returns [[0, 0, "point0,0,0"], [1, 1, "point1,1,1"], [2, 4, "point2,2,4"], [3, 9, "point3,3,9"], [4, 16, "point4,4,16"], [5, 15, "point5,5,25"]]
getVertices(text, ",", ";", 1, 2, true);
